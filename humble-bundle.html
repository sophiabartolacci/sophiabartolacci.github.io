Humble Bundle Redesign

View the design on Figma!
Overview
  • Humble Bundle's website faced significant usability challenges that were hindering the user experience. The cluttered navigation, inconsistent filtering system, and disruptive promotional banner created visual overload and made it difficult for users to find and purchase digital content effectively.
  • I redesigned the website to address these pain points through comprehensive user research and strategic design improvements. My approach included conducting card sorting exercises with three participants to understand how users naturally organize information, followed by applying established UX heuristics to create a more intuitive site structure.
  • The redesign resulted in a significantly cleaner and more user-friendly experience that reduced cognitive load while maintaining Humble Bundle's charitable mission and visual appeal. The new design improved content discoverability, streamlined the shopping process, and created a more cohesive experience for the platform's tech-savvy customer base who value both digital content and charitable giving.
Research & Process
  • User Testing: Conducted card sorting exercises with three users to understand navigation preferences, revealing insights from minimalist to detailed organizational approaches.
  • Heuristic Evaluation: Applied Nielsen's usability principles to identify key issues including redundant promotional elements and inconsistent visual hierarchy.
  • Information Architecture: Redesigned site structure balancing simplicity with depth, incorporating user feedback to create intuitive navigation paths.
  • Visual Design: Maintained Humble Bundle's clean aesthetic while improving contrast, spacing, and content organization to reduce visual clutter.
  • Responsive Design: Ensured consistent experience across devices with improved mobile navigation and touch-friendly interface elements.
Key Improvements
  • Streamlined Navigation: Removed redundant promotional banner and reorganized menu structure based on user card sorting results.
  • Enhanced Store Layout: Redesigned cluttered store page with consistent filtering options and improved content presentation to highlight key elements.
  • Improved Visual Hierarchy: Applied consistent color scheme and spacing to reduce cognitive load and make colorful bundle covers stand out effectively.
  • Optimized Filtering: Implemented unified filter system across all pages, allowing users to sort by popularity, content type, and release status seamlessly.
  • Accessibility Focus: Enhanced contrast ratios, improved keyboard navigation, and ensured screen reader compatibility throughout the redesign.
